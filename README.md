# homebrew-zpl-toolchain

Homebrew tap for zpl-toolchain.

## Install

```bash
brew tap trevordcampbell/zpl-toolchain https://github.com/trevordcampbell/homebrew-zpl-toolchain
brew install trevordcampbell/zpl-toolchain/zpl-toolchain
```

## Notes

- Formula updates are automated from the main zpl-toolchain release workflow.
- Canonical docs: https://github.com/trevordcampbell/zpl-toolchain/blob/main/docs/HOMEBREW.md
